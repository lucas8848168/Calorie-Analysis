## 响应式布局

### 桌面端 (>768px)

- 食物卡片: 4列网格布局
- 卡片尺寸: 140px x 180px
- 间距: 12px

### 平板端 (480px - 768px)

- 食物卡片: 3列网格布局
- 卡片尺寸: 120px x 160px
- 间距: 10px

### 移动端 (<480px)

- 食物卡片: 2列网格布局
- 卡片尺寸: 自适应
- 间距: 8px
- 最近食用列表: 垂直堆叠

## 颜色方案

```
主色调:
- 主色: #667eea (紫蓝色)
- 渐变: linear-gradient(135deg, #667eea 0%, #764ba2 100%)

辅助色:
- 边框: #e0e0e0
- 背景: #ffffff
- 文字: #333333
- 次要文字: #666666
- 提示文字: #999999

状态色:
- 悬停: rgba(102, 126, 234, 0.05)
- 激活: #667eea
- 删除: #f44336
```

## 动画效果

1. **卡片悬停**: 
   - 上移 2px
   - 阴影加深
   - 边框变色
   - 过渡时间: 0.3s

2. **按钮点击**:
   - 缩放 0.98
   - 过渡时间: 0.2s

3. **标签页切换**:
   - 淡入淡出
   - 过渡时间: 0.3s

4. **删除按钮**:
   - 透明度 0 → 1
   - 过渡时间: 0.3s
